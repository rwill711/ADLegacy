# Browser Testing Guide

## Quick Start

Simply **double-click** `TacticalRPG.html` to play the game in your browser!

## If Double-Click Doesn't Work

Try these options:

### Option 1: Right-Click → Open With
1. Right-click on `TacticalRPG.html`
2. Select "Open With"
3. Choose your browser (Chrome, Firefox, Edge, Safari)

### Option 2: Drag and Drop
1. Open your web browser
2. Drag `TacticalRPG.html` into the browser window

### Option 3: Use File → Open
1. Open your web browser
2. Press `Ctrl+O` (Windows/Linux) or `Cmd+O` (Mac)
3. Navigate to and select `TacticalRPG.html`

### Option 4: From Command Line
```bash
# Windows
start TacticalRPG.html

# Mac
open TacticalRPG.html

# Linux
xdg-open TacticalRPG.html
```

## Requirements

- Any modern web browser (Chrome, Firefox, Edge, Safari)
- Internet connection (only needed once to download libraries from CDN)

## Files

- `TacticalRPG.html` - Standalone game file (double-click to play)
- `index.html` - Alternative version (same content)
- `TacticalRPG (2).jsx` - Original JSX source file

## Troubleshooting

If the game doesn't load:
1. Make sure you have an internet connection
2. Try a different browser
3. Check browser console (F12) for errors
4. Disable browser extensions that might block scripts
